## Project build

This project requires [bun.sh](https://bun.com) or Node.js.
After installation complete, simply just run:

```bash
bun i
bun build
bun preview
```

or

```bash
npm i
npm build
npm preview
```
Thats it.

## Accessing CMS & edit the posts

The whole project was relying on TinaCMS & TinaCloud.
A TinaCloud account will be required.

After login, simply just go to: `https://sdp.org.tw/admin`, then you'll get into admin dashboard.
